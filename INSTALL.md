# HOW TO INSTALL THE SUSE

1. Set image up on nebula
2. Assign 2 interfaces for network (write the ID's)
3. Setup the network config.
4. Setup the DHCP server
5. Setup the firewall
6. Install Docker
7. Docker pull mitmproxy/mitmproxy
